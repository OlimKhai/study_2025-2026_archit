## Структура репозитория
- `labs/` - лабораторные работы (код + отчёты)
- `docs/` - методички, требования, ТЗ
- `assets/` - схемы, скриншоты, диаграммы
- `utils/` - вспомогательные скрипты и шаблоны

## Как использовать
Каждая работа находится в `labs/labXX/`:
- `src/` - исходный код / скрипты
- `docs/` - черновики отчёта
- `report.md` или `report.pdf` - финальный отчёт
- `README.md` - как запустить и что делает работа

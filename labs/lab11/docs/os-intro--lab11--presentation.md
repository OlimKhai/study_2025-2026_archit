# ✂️ Редактирование текста

**Вырезание строки:** Команда `C-k`  
**Вставка:** Команда `C-y`

![Процесс вырезания строки](/home/ofkhayjdarovdk5n18/Pictures/Screenshots/editing1.png)

---

# 📋 Копирование текста

1. Выделяем область: `C-space`
2. Копируем в буфер: `M-w`
3. Вставляем: `C-y`

![Выделение и копирование](/home/ofkhayjdarovdk5n18/Pictures/Screenshots/copying1.png)

---

# 🪟 Управление окнами

Разделяем экран на 4 части:
1. По вертикали: `C-x 3`
2. По горизонтали: `C-x 2`

В каждом окне открываем свой буфер (test1.txt, test2.txt и т.д.)

![Разделение окон на 4 части](/home/ofkhayjdarovdk5n18/Pictures/Screenshots/split1tab.png)

# 🔍 Поиск и замена

**Поиск (`C-s`):**
Вводим слово внизу, Emacs подсвечивает результат.

**Замена (`M-%`):**
Вводим старое слово → новое слово. Используем `!` для замены всего сразу.

![Поиск текста](/home/ofkhayjdarovdk5n18/Pictures/Screenshots/search.png)

# 🔧 Режимы работы редактора vi

| Режим | Назначение |
|-------|-----------|
| **Командный** | Ввод команд редактирования и навигации |
| **Вставки** | Ввод содержания файла |
| **Последней строки** | Запись изменений и выход из редактора |

> 💡 Переключение: `Esc` → командный режим, `:` → последняя строка

---

# 🧭 Команды навигации

h ←  |  l →  |  k ↑  |  j ↓     — перемещение курсора
0    — начало строки      $ — конец строки
G    — конец файла        nG — строка №n
w/b  — по словам          /текст — поиск

---

# ✏️ Основные команды редактирования

**Вставка:**
- `i` — перед курсором | `a` — после | `o` — новая строка снизу

**Удаление:**
- `x` — символ | `dw` — слово | `dd` — строка

**Копирование/вставка:**
- `yy` — копировать строку | `p` — вставить

**Отмена:**
- `u` — отменить | `.` — повторить

❓ Контрольные вопросы (краткие ответы)

    3 режима: командный, вставки, последней строки
    Выход без сохранения: :q!
    Позиционирование: 0, $, G, nG
    Слово: зависит от регистра команды (w/W)
    Начало/конец файла: gg / G
    Группы команд: вставка, удаление, копирование, поиск
    Заполнить строку $: режим вставки + повтор .
    Отмена: u
    Последняя строка: :wq, :n,m d, :i,j t k
    Конец строки: $ или :set list
    Опции: :set all, :set <имя>
    Определить режим: индикатор -- INSERT -- или :
    Граф режимов: см. схему в отчёте

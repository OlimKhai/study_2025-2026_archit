Контрольные вопросы (1-5)

  1. Командная оболочка — программа для взаимодействия с ОС

Примеры: sh, csh, ksh, BASH

  1. POSIX — стандарты для совместимости UNIX/Linux систем

  2. Переменные: имя=значение, массивы: set -A имя val1 val2

  3. let — арифметика, read — ввод с клавиатуры

  4. Операции: +, -, *, /, %, &&, ||, <, > и др.

  1. (()) — арифметические выражения без let

  2. Стандартные переменные: PATH, HOME, IFS, MAIL, TERM

  3. Метасимволы: *, ?, [], <, >, |

  4. Экранирование: , ’’, “”

  5. Создание файлов: создать → chmod +x → запуск

  1. Функции: fname() { команды }

  2. Проверка файлов: test -f, test -d

  3. set — список переменных, typeset — атрибуты, unset — удаление

  4. Параметры: 0, $1, $2, … ${10},

  5. Специальные переменные: ?,?,,,!, ∗,∗,@
